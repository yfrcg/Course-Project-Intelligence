# Safety Policy

## 项目定位

`Course Project Intelligence MCP Server` 是一个学习参考和课程项目情报工具。

它的目标是帮助用户：

- 检索公开课程项目与仓库
- 比较不同项目路线与模块拆分
- 提取公开页面的项目摘要
- 汇总公开学习资源和风险提示

## 非目标

本项目不是：

- 代写工具
- 自动生成课程作业答案的工具
- 可直接提交的作业生成器
- 现成答案聚合器

## 允许的使用方式

- 参考公开仓库的模块划分
- 对比不同项目技术路线
- 阅读 README、报告结构和运行说明
- 调研课程相关公开资料和学习资源
- 识别潜在风险，如资料过时、仓库已归档、provider 数据不完整

## 不允许的方向

- 生成可直接提交的作业内容
- 鼓励复制粘贴公开项目作为个人课程提交物
- 把公开仓库重组后伪装成原创作业
- 提供绕过课程规则、评分规则或学术诚信要求的建议

## 输出约束

当前项目通过以下方式维持边界：

- tool 结果中包含 `risk_note`
- 顶层结果中包含 `safety_note`
- README 和文档明确声明仅用于学习参考
- 排名和摘要聚焦公开资料发现、项目结构理解和路线比较

## 贡献要求

所有贡献者都应保持这一边界：

- 可以增强检索、排序、解释和部署
- 可以增强 provider、评测和测试
- 不应新增代写、作业生成、答案拼接相关能力

如果某项改动会模糊“学习参考工具”和“代写工具”的边界，则不应合入。
