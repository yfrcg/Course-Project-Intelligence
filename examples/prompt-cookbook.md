# Prompt Cookbook

This cookbook is GitHub-only.

Use all results as learning references only. They are not official course materials, and they must not be copied directly for submission.

## Trigger `search_course_projects`

- `帮我找 GitHub 上适合作为数据库课程设计参考的公开仓库`
  Recommended tool: `search_course_projects`
- `Find public GitHub repositories for university operating system labs`
  Recommended tool: `search_course_projects`
- `Search GitHub repositories for compiler lab reports, src, and notes`
  Recommended tool: `search_course_projects`
- `Look for public GitHub repositories for Java Web course projects`
  Recommended tool: `search_course_projects`

## Trigger `search_course_resources`

- `帮我找 GitHub 上的数据库课程 labs、assignments、notes 和 reports`
  Recommended tool: `search_course_resources`
- `Find GitHub course materials for university compiler labs`
  Recommended tool: `search_course_resources`
- `Search public GitHub repositories for operating system course notes and lab materials`
  Recommended tool: `search_course_resources`
- `Look for GitHub-hosted course materials for computer networks labs`
  Recommended tool: `search_course_resources`

## Trigger `inspect_course_project`

- `Analyze this GitHub repository as a learning reference for a Java Web course project`
  Recommended tool: `inspect_course_project`
- `Inspect this GitHub repo and tell me whether it contains report, sql, schema, or src assets`
  Recommended tool: `inspect_course_project`
- `看看这个 GitHub 仓库适合参考哪些部分`
  Recommended tool: `inspect_course_project`
- `Check whether this GitHub repository has README, notes, lab, or report materials`
  Recommended tool: `inspect_course_project`

  Recommended tool: `build_course_context`
- `Turn these inspect results into a final agent context pack`

- `Collect evidence cards before answering a course project question`

## Non-GitHub Rewrite

- Not recommended: `分析这个学校课程网页`
  Better: `Please provide a GitHub repository URL for the course project, then inspect it as a learning reference`
- Not recommended: `看看这篇博客能不能参考`
  Better: `Find public GitHub repositories for the same course topic and build an Evidence Pack`
